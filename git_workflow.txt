1. Initialize the Git repository
   git init

2. Stage all project files for commit
   git add .

3. Commit the changes with a descriptive message
   git commit -m "Initial commit: Househunt project with signup, email, user list, and workflow doc"

4. Create a remote repository on GitHub named 'househunt'

5. Link the local repository to the remote
   git remote add origin https://github.com/Kamure/househunt.git

6. Rename the default branch to 'main'
   git branch -M main

7. Push the project to GitHub
   git push -u origin main

8. After updates, repeat:
   git add .
   git commit -m "Describe your update here"
   git push 

  - you can also use the source control in vscode to do step 7 and 8
